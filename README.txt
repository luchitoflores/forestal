
Como usar el "Sistema para Análisis de Usos Potenciales de la Madera"
=====================================================================

Si usa el Sistema Operativo Windows:
-----------------------------------------------------------------------

 * Ejecute el programa "forestweb" que se encuentra en su escritorio y en el menú "Todos los Programas" 
 * La "documentación" y el "manual" con capturas de pantalla se encuentran el directorio c:\Forestal\doc
 * NOTA: Si en algun momento tiene problemas para iniciar la aplicacion "forestweb":
      - Elimine el archivo c:\forestal\mongodb\win32\mongod.lock 
      - Ejecute el programa "forestalini" del menu "Todos los programas" > "Inicio".
